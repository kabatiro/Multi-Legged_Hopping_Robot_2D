## Overview
関節にダンパーを入れた2本脚ロボット（2次元）を上から落とした時の挙動をシミュレーションする

## Requirement
- Simscape Multibody

## Usage
hoppingrover.slxを起動することでSimscapeの画面が開く．
plot_velocity.mでローバーの速度の時間変化をプロットすることができる．
10秒後の速度がコマンドラインに表示される
:zap:
